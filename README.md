# patch-hth-lab

一个用于归档和发布独立 HTML 页面的仓库。  
本仓库不针对任何特定网站或域名，仅作为静态 HTML 内容的存放与展示空间。

## 项目简介

patch-hth-lab 主要用于收集、整理和发布一些独立的 HTML 页面。这些页面可能包含小工具、演示、文档或实验性内容，适合在浏览器中直接打开查看。

仓库结构简单，每个 HTML 文件代表一个独立的页面，彼此之间没有依赖关系。

## 目录说明

```
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面引用的静态资源（如 CSS、JS、图片等）
├── index.html      # 仓库首页/导航页
└── README.md       # 本说明文件
```

- `pages/`：核心目录，每个 `.html` 文件即为一个独立页面。
- `assets/`：公共资源目录，页面如需外部样式或脚本，建议统一放置于此。
- `index.html`：可选的导航页，用于快速浏览所有归档页面。

## 页面归档说明

- 每个页面均保持独立，不依赖其他页面。
- 页面内容可能包含前端演示、信息展示、简单交互等。
- 所有页面均为静态 HTML，无需后端服务即可在浏览器中运行。
- 部分页面可能引用外部资源（如字体、库文件），建议保持网络通畅。

## 维护说明

- 欢迎提交新的独立 HTML 页面。
- 新页面请放入 `pages/` 目录，并确保文件名简洁、无特殊字符。
- 如页面有引用资源，请一并放入 `assets/` 对应子目录。
- 本仓库不提供自动部署或 CI 配置，页面更新后需手动推送。
- 不保证页面兼容所有浏览器，建议使用现代浏览器查看。
